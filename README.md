# quote-api
Just a simple Node.js/Express RESTful API to Create, Read, Update, or Delete (CRUD operations) quotes within a MongoDB (full of quotes... when I add more in).

### What to expect from a HTTP GET Request
```json
{
  "quoteAuthor": "Author's Name",
  "quoteText": "Text of the quote",
  "quoteAccessedDate": "Date that the quote was created"
}
```
